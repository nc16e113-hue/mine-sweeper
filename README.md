# マインスイーパー

定番の地雷さがしパズル「マインスイーパー」を無料で遊べるWeb版です。

- 完全クライアントサイド（静的HTML1枚・バックエンド/API不要・データ収集なし）
- 3つの難易度（やさしい9×9・ふつう12×12・むずかしい13×13）
- 最初のタップは必ず安全（first-click safe）、フラグモード切替（スマホ対応）＋PCは右クリック
- タイム計測・難易度別ベストタイム記録・結果をXにシェア
- 初回に「あそびかた」を自動表示（?ボタンでいつでも再表示）

「AI CEO」プロジェクト（[note.com/ai_ceo_kei](https://note.com/ai_ceo_kei)）が運営するカジュアルゲーム。エンタメ・クラスタのゲーム第2弾です（第1弾は[ことばアタック](https://nc16e113-hue.github.io/kotoba-attack/)）。
